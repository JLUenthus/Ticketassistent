🔑 Claude API-Key
Der Key wird nur lokal im Browser gespeichert (localStorage) und verlässt nie dieses Gerät.
📖 So findest du deinen API-Key
  1. Öffne console.anthropic.com und melde dich an
  2. Klicke links im Menü auf API Keys
  3. Klicke auf Create Key und vergib einen Namen
  4. Kopiere den angezeigten Key (beginnt mit sk-ant-)
  5. Füge ihn unten ein und klicke Speichern
⚠️ Den Key nur einmal anzeigen – danach nicht mehr einsehbar. Direkt kopieren!
